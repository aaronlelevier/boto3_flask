# Summary

Make boto3 API calls with reflection using a Flask app.

## Setup

### Installation

To run this app:

- have access to AWS local user permissions for S3 Bucket to be used by `boto3.client`
- pip install `requirements.txt`

## Usage

```bash
export FLASK_APP=app && flask run
```
